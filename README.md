Interactive geospatial visualization platform built with React, Vite, Cesium, and Resium. Supports live vessel tracking, heatmaps, drawing tools, layered imagery, and report generation.

Deployed to www.dan-sapp.com

- 🌍 3D globe visualization with Cesium
- 🚢 Real-time vessel tracking
- ✏️ Drawing and annotation tools
- 🔥 Heatmap overlays
- 📡 Layered imagery support
- 📱 Responsive side panels
- ⚡ Fast Vite-based development

## Tech Stack

- React
- TypeScript
- Vite
- Cesium
- Resium
- TailwindCSS
- Redux Toolkit / RTK Query

## Getting Started

### Install

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build

```bash
npm run build
```
