- 🚢 Real-time vessel tracking
- ✏️ Drawing and annotation tools
- 🔥 Heatmap overlays
- 📡 Layered imagery support
- ⚡ Fast Vite-based development

## Tech Stack

- React
- TypeScript
- Vite
- Cesium
- Resium
- TailwindCSS
- Redux Toolkit / RTK Query

## Getting Started

### Install

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build

```bash
npm run build
```
